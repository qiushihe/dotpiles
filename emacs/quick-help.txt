Rectandle operations

* Rectandles marked by start and
  end of selected region
* C-x r o : Insert space (shift text
  to the left)
* C-x r k : Delete rectangle
* C-x r c : Replace with space
* C-x r t : Replace with text

Adjust indentation level

1. Select a block of text
2. C-x <tab>
3. Use left/right arrows
